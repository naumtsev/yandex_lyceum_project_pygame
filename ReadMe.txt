Игра «Звёздные бои»

Данное приложение является проектом для Яндекс.Лицея
В разработке использовалась библиотека pygame.

Игра предназначена для двух игроков.
Необходимо сделать так, чтобы шар ударился о сторону противника.
В игре присутствуют блоки, которые мешают игрокам отбивать мяч в сторону противника.
Чтобы усложнить игру, достаточно увеличить скорость мяча.
Это можно сделать в исходном файлу solution.py.
